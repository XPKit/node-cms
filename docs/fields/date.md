# Date Field

The `date` field type is used for date input.

## Declaration
```js
{
  field: 'date',
  input: 'date'
}
```

## Usage
- Used for selecting a date (no time).

## Options
| Option      | Type    | Description                                 |
|-------------|---------|---------------------------------------------|
| label       | String/Object | Field label (can be localized)         |
